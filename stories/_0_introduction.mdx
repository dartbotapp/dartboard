```bash
npm install @dartbot/dartboard
```
```html
<script type="module">
  import '@dartbot/dartboard/dartbot-dartboard.js';

```

# Features
* Styles
  * ✅ Customize styles via CSS
    * ✔️ Show wire
    * ✔️ Wire shadow
    * ✔️ Wire blur
    * ✔️ Wire offset
    * ✔️ Wire overhang
    * ✔️ Wire inset
    * ✔️ Show numbers
  * ✅ Support standard CSS props
    * ✔️ padding
    * ✔️ margin
    * ✔️ filters
    * ✔️ transformations
    * ✔️ zoom
  * ✅ Resize
* Hits✅
* Alt board setups✅
* Events
  * hover
  * click
  * select
  * drag
  * touch
* Pan and Zoom
  * props
  * mouse
* Npm build and publish
* Unit tests
* Build badges
* Rotate board to match pi
